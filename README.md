# 🪙 Coin Tracker

Aplicativo desktop para fazer consulta e conversão entre moedas.

---

## Tecnologias
- Java 21
- JavaFX
- Maven

## Como executar

1. Clone o projeto

````bash
git clone https://github.com/filipemartinsdev/coin-tracker.git
cd coin-tracker
````

2. Execute a aplicação

````bash
mvn javafx:run
````
